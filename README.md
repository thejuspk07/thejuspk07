### 🏥 Medical RAG Pipeline *(Capstone — WIP)*
> Explainable AI for medical reports using retrieval-augmented generation

- Hybrid retrieval: **BM25 sparse + dense vector search** fused with Reciprocal Rank Fusion (RRF)
- Custom document chunker optimized for clinical text structure
- **NLI-based hallucination detection** to verify LLM output against retrieved context
- PubMed fine-tuned embeddings for domain-accurate retrieval
- LLM used only as final generation step — core intelligence stays in the retrieval layer
- **Stack:** Python · LangChain · FAISS · HuggingFace Transformers · BM25Okapi · FastAPI

*Research target: EMNLP 2025 / NeurIPS workshop*
