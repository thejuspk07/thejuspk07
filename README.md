## 🚀 Projects

### 🔧 LSTM-Based Multi-Sensor Anomaly Detection
> AI-powered motor health monitoring with real-time anomaly detection

- Designed and built an **LSTM Autoencoder** to detect anomalies from live multi-sensor data streams (vibration, gyroscope, temperature)
- Integrated **MPU6050 accelerometer/gyroscope** via Arduino with serial communication for real-time data acquisition
- Built **Flask + Streamlit dashboards** for live reconstruction error visualization and monitoring
- Implemented **Arduino LED alert system** triggered on anomaly detection
- Full pipeline: sensor → preprocessing → LSTM training → real-time inference → dashboard → alert

**Stack:** Python · TensorFlow · Keras · Flask · Streamlit · Arduino · NumPy · Pandas

### 🐦 Sentiment Analysis of Tweets
> Real-time tweet sentiment classification using NLP and SVM

- Built a complete **NLP pipeline**: text cleaning → TF-IDF vectorization → SVM classification → prediction
- Trained on labeled tweet datasets to classify sentiment as positive or negative
- Deployed as an **interactive Streamlit web app** with real-time prediction and sentiment visualization
- Implemented evaluation metrics (accuracy, confusion matrix) for model performance reporting

**Stack:** Python · Scikit-learn · TF-IDF · SVM · Streamlit · NLTK · Pandas
